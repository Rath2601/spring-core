### 1. IoC Container Deep Dive in Spring
#### IoC Packages
- `org.springframework.beans`
- `org.springframework.context`

#### BeanFactory
- Basic IoC container
- Lazy initialization
- Low-level API

#### ApplicationContext
- Superset of BeanFactory
- Eager initialization
- AOP support
- Event publishing
- Internationalization
- Web support (`WebApplicationContext`)
